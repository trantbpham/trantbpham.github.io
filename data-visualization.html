Data Visualization

This data represents the average class size for each district in New York City (NYC) for grades K-9.

This shows the average class size is around 25 students per class, with each school has around 50 sections per school.

We're looking at the commonality in average class size vs. section across schools in NYC, in order to understand if

this is the optimal solution for students, and if there should be additional support for the schools if the class size

versus number of sections is an issue for students learning.
